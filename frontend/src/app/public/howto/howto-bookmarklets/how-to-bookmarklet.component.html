Bookmarklets

Bookmarklets are little javascript links that live in the bookmarks toolbar of your browser.

We recommend you drag one of the links bellow to your browser's toolbar

  • Bookmarks

    • +Bookmarks.dev - popup - when clicked a popup to save the current URL is displayed [recommended for desktops and notebooks with good resolution]
    • +Bookmarks.dev - go back - when clicked it redirects you to the New Bookmark form. Once added you go back to the initial page [recommended for phones or smaller resolution devices]

    You can select the text on page that you want to be added to the description of the bookmark before you click the bookmarklet's link

    To save regular bookmarks you can also use the Save to Bookmarks.dev extension for Chromium based browsers. It provides the same behaviour as the first bookmarklet.

  • Codelets

    • +Codelet - popup - when clicked a popup to save the current URL is displayed [recommended for desktops and notebooks with good resolution]
    • +Codelet - go back - when clicked it redirects you to the New Codelet Form in the same window. Once added you go back to the initial page [recommended for phones or smaller resolution devices]

    For Codelets you should select prior the code snippet from the page you are surfing on. This will be "pasted" in the New Codelet Form, otherwise it makes less sense to use the bookmarklet
