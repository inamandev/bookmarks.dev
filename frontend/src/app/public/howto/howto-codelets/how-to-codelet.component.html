Codelets

Codelets are special types of bookmarks where you save small pieces of code you would want to recall later. At the present moment all codelets are private only.

Save codelets

You need to select the Codelet option in the menu bar
Bookmark Codelet

Search codelets

Searching for codelets works the same as the one for regular bookmarks. Just select the My codelets option in the search bar. You can save searches the same way as for regular bookmarks.
